
# Zilch

Dice game just involving 6 dice and a pencil and paper to keep score. In the process of making a barebones version.


## Scores:

| DICE HAND   | SCORE           |
| ----------- | --------------- |
| 1           | 10              |
| 5           | 5               |
| 3 of a kind | dice value x 10 |
| 4 of a kind | dice value x 20 |
| 5 of a kind | dice value x 40 |
| 5 of a kind | 1000            |
| straight    | 300             |
| 3 pair      | 100             |
